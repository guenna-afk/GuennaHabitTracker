# Guenna Tracker 📊

**Guenna Tracker** es una herramienta integral de gestión personal diseñada para organizar y visualizar tu progreso mensual en dos áreas fundamentales: **Hábitos** y **Finanzas**. Con una interfaz intuitiva y limpia, permite un seguimiento detallado para alcanzar tus objetivos personales y mantener tu salud financiera bajo control.

## 🚀 Características Principales

### 1. Panel de Hábitos (Habit Tracker)
Seguimiento diario y semanal para construir una disciplina sólida.
* **Seguimiento Diario:** Visualización tipo cuadrícula para los 31 días del mes.
* **Seguimiento Semanal:** Control de objetivos que se cumplen por semana (S1 a S5).
* **Métricas de Progreso:**
    * % de Progreso Global.
    * Contador de hábitos completados y restantes.
    * Cálculo de la "Mejor Racha" para mantener la motivación.
* **Ranking:** Top de hábitos del mes basados en cumplimiento.

### 2. Panel de Finanzas (Finance Tracker)
Gestión inteligente de flujos de caja y distribución de gastos.
* **Balance Automático:** Visualización en tiempo real del saldo neto (Ingresos - Gastos).
* **Categorización de Gastos:**
    * **Gastos Fijos:** Alquiler, suscripciones, facturas, etc.
    * **Gastos Variables/Puntuales:** Compras ocasionales, ocio, emergencias.
* **Distribución Visual:** Gráficos para entender en qué se va el dinero.
* **Historial Detallado:** Registro por concepto, categoría, fecha y cantidad.

### 3. Personalización y Tiempo
* **Selector de Fecha:** Soporta un rango desde el año 2020 hasta el 2035.
* **Selector Mensual:** Navega fácilmente entre los meses del año para ver registros históricos o planificar el futuro.
* **Botón de Reinicio:** Limpia los datos para comenzar un nuevo ciclo con un solo clic.

## 🛠️ Tecnologías Sugeridas (Opcional si es un proyecto de código)
* **Frontend:** HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+).
* **Almacenamiento:** LocalStorage para persistencia de datos local o integración con base de datos (Firebase/Supabase).

## 🖥️ Vista Previa de la Interfaz

### Panel de Control
El diseño está dividido en una barra lateral de navegación y un área de trabajo principal que cambia según la sección seleccionada (Hábitos o Finanzas).

| Sección | Descripción |
| :--- | :--- |
| **Dashboard** | Vista general del mes y año seleccionado. |
| **Hábitos** | Cuadrícula de 1-31 días y resumen semanal. |
| **Finanzas** | Tablas de ingresos y desglose de gastos. |
